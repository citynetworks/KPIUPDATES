Connect to Google Sheets

To load live data, you need to deploy a small Google Apps Script in your spreadsheet. This is a one-time setup that takes about 2 minutes.

1
Open your Google Sheet

Go to your City Network spreadsheet ↗, then click Extensions → Apps Script in the top menu.

2
Paste this code

Select all the existing code in the editor and replace it with the code below, then click Save (Ctrl+S). If you already deployed a previous version, you'll need to redeploy with this updated code.

const SPREADSHEET_ID = '1aRP9xWFsqOoh-zAXKI4olsCjn29Q_sf-1xKsHUptLQs'; const SHEET_NAMES = ['Full Time Rep - Install', '1099 B - Rep Install', 'CN Installs - Onboarding']; function doGet(e) { try { const ss = SpreadsheetApp.openById(SPREADSHEET_ID); const result = {}; SHEET_NAMES.forEach(function(name) { const sheet = ss.getSheetByName(name); result[name] = sheet ? sheet.getDataRange().getValues() : []; }); const json = JSON.stringify({ ok: true, data: result }); const cb = e && e.parameter && e.parameter.callback; const output = ContentService.createTextOutput(cb ? (cb + '(' + json + ')') : json); output.setMimeType(cb ? ContentService.MimeType.JAVASCRIPT : ContentService.MimeType.JSON); return output; } catch(err) { const json = JSON.stringify({ ok: false, error: err.toString() }); const cb = e && e.parameter && e.parameter.callback; const output = ContentService.createTextOutput(cb ? (cb + '(' + json + ')') : json); output.setMimeType(cb ? ContentService.MimeType.JAVASCRIPT : ContentService.MimeType.JSON); return output; } }
3
Deploy as a Web App

Click Deploy → New deployment. Set type to Web app. Under "Execute as" choose Me. Under "Who has access" choose Anyone. Click Deploy and authorize when prompted.

4
Paste the Web App URL below

After deploying, copy the Web App URL (it looks like https://script.google.com/macros/s/…/exec) and paste it here.
